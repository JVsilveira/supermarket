# SuperMercado

Aplicação web desenvolvida com HTML, CSS e JavaScript puro, simulando a interface de um supermercado online. O projeto foi criado com o objetivo de praticar conceitos fundamentais de desenvolvimento Front-End, incluindo responsividade, manipulação do DOM, organização de componentes visuais e interatividade utilizando JavaScript.

## Demonstração

O sistema apresenta uma página de supermercado contendo:

- Menu de navegação responsivo
- Carrossel automático de ofertas
- Campo de pesquisa de produtos
- Catálogo de produtos em grade
- Menu hamburguer para dispositivos móveis
- Feedback visual para pesquisas sem resultados
- Botão de retorno ao topo da página
- Layout responsivo para diferentes tamanhos de tela

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## Funcionalidades

### Carrossel de Ofertas

- Navegação manual através dos botões de avançar e voltar
- Rotação automática dos produtos em destaque
- Pausa automática ao posicionar o cursor sobre o carrossel
- Reinício automático da navegação após interação do usuário

### Pesquisa de Produtos

- Busca em tempo real
- Filtragem dinâmica dos itens exibidos
- Exibição de mensagem quando nenhum produto é encontrado
- Atualização instantânea dos resultados durante a digitação

### Menu Responsivo

- Menu hamburguer para dispositivos móveis
- Abertura e fechamento através de animações suaves
- Navegação simplificada para diferentes seções da página

### Layout Responsivo

- Adaptação automática para desktops, tablets e smartphones
- Grid flexível para exibição dos produtos
- Componentes ajustados para diferentes resoluções

## Conceitos Aplicados

Durante o desenvolvimento foram utilizados diversos conceitos importantes de Front-End:

- Organização semântica do HTML
- CSS Variables
- Flexbox
- CSS Grid
- Responsividade
- Manipulação do DOM
- Programação Orientada a Objetos (POO)
- Classes JavaScript
- Event Listeners
- Debounce visual através de eventos de input
- Boas práticas de acessibilidade
- Lazy Loading de imagens

## Objetivo do Projeto

Este projeto foi desenvolvido para fins de estudo e aperfeiçoamento das habilidades em desenvolvimento Front-End, explorando recursos nativos da plataforma web sem a utilização de frameworks ou bibliotecas externas.
